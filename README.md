# Таблица пользователей

## Цель проекта

Тестовое задание: одностраничное приложение для управления списком пользователей. Таблица с колонками «Имя», «Телефон», «Роль» и действиями; добавление пользователей через модальное окно с валидацией, удаление по кнопке. Данные хранятся в `localStorage`.

## Использованные технологии

- **Vue 3** — фреймворк, Composition API
- **Vite** — сборка и dev-сервер
- **TypeScript** — типизация
- **Pinia** — состояние (список пользователей, добавление/удаление)
- **Tailwind CSS** — стили (tailwind подключен, но используются только scoped стили)

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`, превью: `npm run preview`.
